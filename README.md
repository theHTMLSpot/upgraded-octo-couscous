# FastAPI Backend Examples

This repository contains various backend API examples built with FastAPI. Each example showcases different functionalities such as managing accounts, contacts, to-do lists, and generating random numbers.

## Examples

1. [Accounts Management](#accounts-management)
2. [Contact List](#contact-list)
3. [To-Do List](#to-do-list)
4. [Random Number Generator](#random-number-generator)

## Requirements

- Python 3.7+
- FastAPI
- Pydantic
- Uvicorn

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/yourusername/fastapi-backend-examples.git
    cd fastapi-backend-examples
    ```

2. Create a virtual environment and activate it:

    ```bash
    python -m venv env
    source env/bin/activate  # On Windows, use `env\Scripts\activate`
    ```

3. Install the dependencies:

    ```bash
    pip install fastapi pydantic uvicorn
    ```

## Running the Application

1. Run the FastAPI application with Uvicorn:

    ```bash
    uvicorn main:app --reload
    ```

2. Open your browser and navigate to `http://127.0.0.1:8000` to see the API documentation generated by Swagger UI.

## Accounts Management

### Endpoints

#### Get All Accounts

- **URL:** `/`
- **Method:** `GET`
- **Response:** List of all accounts

#### Get Account by ID

- **URL:** `/account/{account_id}`
- **Method:** `GET`
- **Response:** Single account matching the provided ID

#### Create Account

- **URL:** `/create-account/`
- **Method:** `POST`
- **Request Body:**
    ```json
    {
        "name": "string",
        "password": "string",
        "date_of_birth": "string",
        "location": "string",
        "bio": "string"
    }
    ```
- **Response:** The created account with an assigned ID

#### Update Account

- **URL:** `/update-account/{account_id}`
- **Method:** `PUT`
- **Request Body:** Any fields to update (all fields optional)
    ```json
    {
        "name": "string",
        "password": "string",
        "date_of_birth": "string",
        "location": "string",
        "bio": "string"
    }
    ```
- **Response:** The updated account

#### Delete Account

- **URL:** `/delete-account/{account_id}`
- **Method:** `DELETE`
- **Response:** The deleted account
